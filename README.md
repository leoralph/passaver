# PASSAVER

Passaver é uma ideia que eu tive, após enfrentar diversos problemas com armazenamento de senhas e arquivos, por diversas vezes esquecendo as minhas senhas, portanto tive essa ideia de criar uma aplicação para guardá-las, a versão original não utilizava o ReactJS, portanto recriei a aplicação inteiramente do zero utilizando a biblioteca React, com o back-end feito em Laravel
